Project Overview

What is Ganesh Transport?

Ganesh Transport Management System is a comprehensive web application built with PHP and MySQL that helps transport companies manage their daily operations efficiently.

Key Objectives

  • Streamline truck and driver management
  • Track trip details and fuel consumption
  • Calculate trip costs and profits
  • Provide user authentication and security
  • Generate reports and analytics

Target Users

Transport Company Owners

Manage multiple trucks and track overall business performance

Fleet Managers

Monitor daily operations and driver assignments

Administrators

Handle system configuration and user management

Core Features

User Authentication

  • Secure login/logout system
  • Session management
  • User registration
  • Password protection

Truck Management

  • Add multiple trucks per user
  • Track truck numbers
  • Driver assignment
  • Truck performance history

Trip Management

  • Record trip details (from/to locations)
  • Track distance and fuel consumption
  • Calculate trip costs
  • Store trip remarks and notes

Financial Tracking

  • Fuel cost calculations
  • Load weight tracking
  • Advance payment management
  • Profit/loss calculations

Database Architecture

Database: ganesh

📋 user
  • sno (int, PK, Auto Increment)
  • email1 (varchar(255), Unique)
  • pass1 (varchar(255))
  • created_at (timestamp)
🚛 trucknum
  • id (int, PK, Auto Increment)
  • truck_num (varchar(50))
  • user_id (int, FK to user.sno)
  • created_at (timestamp)
📊 transport
  • id (int, PK, Auto Increment)
  • truckuser_id (int, FK to user.sno)
  • truck_driver (varchar(100))
  • truck_id (varchar(50))
  • date (date)
  • from/to (varchar(255))
  • start/end (decimal(10,2))
  • km/ltr/amt/avg (decimal(10,2))
  • weight/load/bhathu/other (decimal(10,2))
  • adv/deva/actual/os (decimal(10,2))
  • remark (text)
📧 contact
  • id (int, PK, Auto Increment)
  • name (varchar(100))
  • email (varchar(255))
  • subject (varchar(255))
  • message (text)
  • created_at (timestamp)

Setup & Installation

Prerequisites

  • XAMPP (Apache + MySQL + PHP)
  • PHP 7.4 or higher
  • MySQL 5.7 or higher
  • Web browser (Chrome, Firefox, Safari)

Installation Steps

  1. Install XAMPP on your system
  2. Start Apache and MySQL services
  3. Copy project files to htdocs folder
  4. Import database using phpMyAdmin
  5. Configure database connection
  6. Access application via localhost

Database Setup

1. Open phpMyAdmin

2. Create new database: ganesh

3. Import database_setup.sql

4. Verify tables are created

Configuration

Database Connection: Update db.php with your credentials

Default Admin: admin@ganeshtransport.com / admin123

Access URL: http://localhost/php/

File Structure

  • index.php - Login page
  • home.php - Main dashboard
  • data.php - Trip management
  • contact.php - Contact form
  • db.php - Database connection
